# BHAS

This repository contains the **BHAS** solution, built with **.NET Framework 4.7.2** (Visual Studio / MSBuild).

## Projects

- `BHAS/`: ASP.NET MVC web application
- `BHAS.DbFirst/`: database-first data access project
- `ClassLibrary1/`: supporting class library

## Build / run (Visual Studio)

- Open the solution (`BHAS.slnx`) in Visual Studio.
- Restore NuGet packages if prompted.
- Build the solution.
- Set `BHAS` as startup project and run.
